## License Types Reference

### Pexels License
- Free to use for personal and commercial purposes
- No attribution required (but appreciated)
- Can modify and adapt
- Cannot sell unmodified videos
- URL: https://www.pexels.com/license/

### Mixkit License
- Free for commercial and personal use
- No attribution required
- Can modify and adapt
- Cannot redistribute as stock footage
- URL: https://mixkit.co/license/

### Pixabay License
- Free for commercial and personal use
- No attribution required
- Can modify and adapt
- Cannot sell unmodified content
- URL: https://pixabay.com/service/license/

### Creative Commons CC0
- Public domain dedication
- No attribution required
- Free for any use
- Can modify and adapt
- URL: https://creativecommons.org/publicdomain/zero/1.0/

### Creative Commons CC-BY
- Free to use with attribution
- Must credit the author
- Can modify and adapt
- Can use commercially
- URL: https://creativecommons.org/licenses/by/4.0/

### YouTube Standard License
- Requires permission from creator
- Check video description for usage rights
- Contact creator for commercial use
- May require attribution

## How to Fill This Out

After downloading a video using the search tool:

1. **Find the video information**:
   - Check the video source website
   - Note the license type
   - Record the author/creator name
   - Copy the direct URL

2. **Add entry to this file**:
   - Use the template above
   - Fill in all fields
   - Be specific about modifications

3. **Keep it updated**:
   - Update when replacing videos
   - Note any changes or modifications
   - Keep backup of original if modified

## Legal Compliance Checklist

Before using any video, verify:

- [ ] License allows your intended use (personal/commercial)
- [ ] Attribution requirements are met (if any)
- [ ] Modifications are allowed (if you plan to edit)
- [ ] No additional restrictions apply
- [ ] Video is not copyrighted or has proper permissions
- [ ] You have saved license information for future reference

## Commercial Use Notice

If you plan to use Smootie commercially:

1. **Verify all videos allow commercial use**
2. **Check if attribution is required**
3. **Keep records of all licenses**
4. **Consider purchasing commercial licenses if needed**
5. **Consult with legal counsel if unsure**

## Questions?

2. Contact the video creator directly
3. Consult legal counsel for commercial use
4. When in doubt, choose videos with clear CC0 or similar licenses
